Mi proyecto


Descripción: El proyecto siguiente es Ubuntu es una maquina virtual la cual simularemos un sistema operativo diferente a Windows dentro de nuestra máquina por eso se le llama máquina virtual con lo siguiente simularemos Linux, un sistema operativo muy famoso por su código abierto al contrario de Windows, con estilo simulamos Linux para tener el conocimiento de poder ver otro sistema operativo y como funcionaba desde adentro desde el cmd

    Objetivos
  • Simular el funcionamiento de un sistema operativo Linux mediante una máquina virtual.
  • Comprender el funcionamiento interno de un sistema operativo de código abierto.
  • Fortalecer los conocimientos sobre sistemas operativos y virtualización.

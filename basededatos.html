Mi Proyecto


Descripción: El proyecto individual consiste en el análisis y desarrollo de tres ejercicios distintos: la gestión de una biblioteca, el control de inventario en una tienda y el funcionamiento de una red social. Cada caso debe ser documentado de manera completa en un archivo PDF, integrando tanto la parte teórica como la práctica. Para cada ejercicio se requiere elaborar un diagrama entidad-relación (ER) en WorkBench, generar el script correspondiente de la base de datos, crear la base y sus tablas en PHPMyAdmin, visualizar el diagrama en el diseñador de esa herramienta e insertar al menos tres registros por tabla para comprobar su funcionamiento En este proyecto se consiguió fundamentar más adentro que es donde se consigue la base de datos el cual se debería de meterse con conexiones 1 por 1 y que tenga una lógica de por medio

    Objetivos
  • Documentar cada caso de manera completa en un archivo PDF.
  • Elaborar diagramas entidad-relación (ER) utilizando WorkBench.
  • Crear bases de datos y tablas en PHPMyAdmin.
    Visualizar y comprobar la estructura de la base de datos mediante diagramas.
